# Les notifications

## Recevoir des notifications

Vous recevrez automatiquement une **notification** dans les cas suivants :

- Lorsqu'une nouvelle publication est partagée dans un fil thématique auquel vous êtes abonné.e
- Lorsqu'un nouveau collaborateur rejoint le meême compte Cocoom que vous
- Lorsqu'un utilisateur commente l'une de vos publications
- Lorsque qu'un administrateur modère votre commentaire
- Lorsque qu'un administrateur modifie votre publication
- Lorsque qu'un administrateur modère votre publication

## Consulter vos notifications

Pour **accéder à l'ensemble de vos notifications**, cliquez sur l'icône en forme de cloche située en haut à droite de votre écran. Un compteur vous indiquera le nombre de notifications non lues.

![Notifications](/img/fr/guide/web-app/notifications.webp)

## Gérer vos notifications

Vous pouvez :

- cliquez sur une notification pour indiquer qu'elle a été lue
- Marquer toutes les notifications comme lues
- Accéder directement au contenu concerné en cliquant sur le bouton associé à la notification (si disponible)
